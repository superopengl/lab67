# Color Palette

Lab67 uses a bright, playful, kid-friendly light theme.

## Brand Colors

| Name | Hex | Usage |
|---|---|---|
| **Primary Green** | `#43b88c` | Brand color, primary buttons, logo icon, links |
| **CTA Yellow** | `#fcd63c` | Call-to-action buttons |
| **CTA Yellow Shadow** | `#e5be2a` | Raised button bottom shadow |
| **Accent Blue** | `#6ec1e4` | Info accents, gradient endpoints |
| **Accent Purple** | `#7c5cfc` | Secondary accents, gradient starts |
| **Accent Amber** | `#f59e0b` | Warning highlights, step markers |
| **Success Green** | `#61ce70` | Success states, active indicators |

## Text Colors

| Name | Hex | Usage |
|---|---|---|
| **Heading** | `#2d3748` | Page titles, card titles, logo text |
| **Body** | `#718096` | Paragraphs, descriptions |
| **Muted** | `#a0aec0` | Placeholder text, secondary labels |
| **On Dark** | `#fff` | Text on colored/gradient backgrounds |
| **On Dark (secondary)** | `rgba(255,255,255,0.9)` | Subtitles on gradients |
| **On Dark (tertiary)** | `rgba(255,255,255,0.6)` | Footer text |

## Background Colors

| Name | Hex | Usage |
|---|---|---|
| **Surface** | `#fff` | Page background, cards, headers, navbar |
| **Canvas** | `#f7fafc` | Section backgrounds, page canvas (admin) |
| **Hero Gradient** | `#43b88c` -> `#6ec1e4` | Hero section (`linear-gradient(135deg, ...)`) |
| **CTA Gradient** | `#7c5cfc` -> `#6ec1e4` | Bottom CTA section |
| **Login Gradient** | `#e8f8f0` -> `#e8f4fa` | Login page background |
| **Terminal** | `#1a1a2e` | Terminal panel in sandbox |
| **Footer** | `#2d3748` | Page footer |

## Icon Badge Backgrounds

Soft tinted backgrounds behind feature icons on white cards.

| Name | Hex | Paired With |

## Borders

| Name | Hex | Usage |
|---|---|---|
| **Default** | `#e2e8f0` | Header borders, panel dividers |
| **Light** | `#f0f0f0` | Navbar border |

## Shadows

| Name | Value | Usage |
|---|---|---|
| **Card** | `0 4px 24px rgba(0,0,0,0.06)` | Feature cards |
| **Card (elevated)** | `0 8px 32px rgba(0,0,0,0.08)` | Login card, modal cards |
| **Card (subtle)** | `0 2px 12px rgba(0,0,0,0.04)` | Tables, low-emphasis surfaces |
| **Text on Gradient** | `0 2px 8px rgba(0,0,0,0.12)` | Headlines on colored backgrounds |
| **CTA Button** | `0 4px 0 #e5be2a` | Yellow raised buttons (large) |
| **CTA Button (small)** | `0 2px 0 #e5be2a` | Yellow raised buttons (nav-size) |
